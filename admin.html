Backup y persistencia

Descarga el backup antes de hacer redeploys. Si los datos se borran, restáuralos desde Base de datos → Restaurar en el panel lateral.

Persistencia permanente en Coolify
  1. Coolify → tu servicio → pestaña Storages
  2. Añadir almacenamiento → ruta /app/data
  3. Guardar y hacer redeploy → los datos ya no se borrarán nunca
