    Música Ambiente
    Biblioteca de pago — tracks, packs y accesos de usuarios
     Importar desde carpeta

    Escanea una carpeta (y subcarpetas) y crea automáticamente los tracks a partir de los archivos de audio encontrados.
    Los metadatos (título, duración, cover embebido) se extraen de los tags ID3/Vorbis.
    Ruta por defecto: carpeta de uploads del servidor (/data/ambient/tracks).
    Para NAS o ruta externa, añade la ruta en Coolify como variable de entorno AMBIENT_SCAN_PATHS=/ruta/musica.

     Importar carpeta de Google Drive

    Importa todos los archivos de audio de una carpeta de Google Drive de golpe.
    La carpeta debe estar compartida como Cualquiera con el enlace puede ver.
    Requiere la variable de entorno GOOGLE_API_KEY configurada en Coolify.

    Backup y persistencia

    Descarga el backup antes de hacer redeploys. Si los datos se borran, restáuralos desde Base de datos → Restaurar en el panel lateral.

    Persistencia permanente en Coolify
    1. Coolify → tu servicio → pestaña Storages
    2. Añadir almacenamiento → ruta /app/data
    3. Guardar y hacer redeploy → los datos ya no se borrarán nunca
